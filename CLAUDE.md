## プロジェクト状況
LoLの大会向けの配信待機画面をHTML+JS+CSSで構成したいです。
OBSのブラウザ機能でローカルHTML+JSを読み取り、カスタムCSSを入力して動作するようにしたいです。


画面はフルHD(1920x180)で、アニメーションを多用してみていて飽きないような配信待機画面を作りたいです。

配信待機画面には

- 時計 (YY:MM:DDとLIVE表記、アニメーションする何か)
- メインメッセージ (配信開始までしばらくお待ちください、など)
- 大会情報(Nexus Cup 交流大会 19:00より、など)
- BGM情報 (右下に小さく、 Inst.音源 - 騒音のない世界 - きらめくウィークエンド、可能ならアニメ付き)
- チーム情報 (5チーム分をかわるがわる配置)
- おちついたアニメーションのある背景、夜空や宇宙、真夜中の湖などを感じさせる暗くて静かな感じのある全体背景アニメーション、飽きないように1分ぐらいのシナリオがありそれらをループ

これらの情報は可能であれば、OBSの設定で差し替えれるようにしたいです。

### 技術構成

- HTML
- JavaScript
- CSS
- p5.js
- アニメーションや効果、3Dなどの演出で美しく見せるのに必要なライブラリやシェーダーなどがあればWebGLで導入してもよい

## 機能要件
- アニメ付き時計
- メインメッセージ表示
- 大会情報表示
- BGM情報表示 (右下に小さく、可能ならアニメ付き)
- チーム情報 (5チーム分をかわるがわる、アニメで表示)
- 落ち着いた雰囲気のある背景アニメーション、1分間分のシナリオあり
- 表示情報の設定機能
- オプション: League of Legendsの画像の差し込み、背景への透かし

## 非機能要件
- 全体的に見ていて飽きないアニメーション
- 落ち着いた神秘的な雰囲気を持つアニメーション背景
- OBSのブラウザで導入可能
- フルHDを前提とした画面構成
- デプロイ方法や遊び方が記載されたREADME.md
- GitHubで管理するための.gitignore
